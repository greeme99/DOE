# **DOE AI 웹앱 화면(UI/UX) 설계서 (Design Specification)**

**버전**: 1.0

**디자인 컨셉**: Industrial & Seamless (산업 현장의 신뢰감 \+ 모바일의 직관성)

## **1\. 디자인 시스템 (Design System)**

### **1.1 컬러 팔레트**

* **Primary**: \#1E3A8A (Deep Blue) \- 신뢰감, 데이터 분석, 헤더 및 주요 버튼.  
* **Success**: \#10B981 (Emerald Green) \- 최적해 도출, 성공 알림, 승인 버튼.  
* **Danger**: \#EF4444 (Coral Red) \- 유의미하지 않은 결과, 경고, 삭제.  
* **Background**: \#F3F4F6 (Light Gray) \- 눈의 피로를 줄이는 현장용 배경색.

### **1.2 타이포그래피 & 버튼**

* **Font**: Noto Sans KR (가독성 중시).  
* **Buttons**: 최소 높이 48px (장갑 착용 시 터치 고려), 12px 라운드 처리.  
* **Input**: 큰 폰트 크기와 명확한 외곽선으로 입력 가시성 확보.

## **2\. 레이아웃 전략 (Layout)**

* **상단 네비게이션**: 현재 진행 중인 스텝 표시 (1 \~ 6단계 인디케이터).  
* **메인 컨텐츠**: 중앙 집중형 카드 레이아웃.  
* **하단 고정 탭 바**: \[설계\] \[입력\] \[분석\] \[최적화\] \[검증\] \[보고서\] 6개 메뉴 상시 노출.

## **3\. 단계별 상세 화면 설계**

### **\[Step 1\] 실험 설계 (Design)**

* **UI 구성**:  
  * 상단 공정 카테고리 선택 (아이콘 그리드).  
  * 인자별 슬라이더 또는 \+/- 버튼형 입력 카드.  
  * '20런 설계표 생성' 플로팅 버튼.  
* **UX 포인트**: 템플릿 선택 시 인자값(온도, 압력 등)이 자동으로 채워지는 애니메이션 제공.

### **\[Step 2\] 실험 데이터 입력 (Input)**

* **UI 구성**:  
  * 상단에 '현장 QR 스캔' 전용 버튼 (크게 배치).  
  * 실험 번호별 리스트 뷰 \+ 수율 입력 칸.  
  * 상단 진행률 바 (예: 12/20 완료).  
* **UX 포인트**: 입력 성공 시 '띵' 소리와 함께 초록색 체크 표시로 피드백 강화.

### **\[Step 3\] 결과 분석 (Analyze)**

* **UI 구성**:  
  * AI 진단 요약 카드 ("사출 온도가 수율에 가장 치명적입니다").  
  * Plotly 차트 슬라이더 (좌우 스와이프로 Pareto \-\> Normal \-\> Cube 등 전환).  
* **UX 포인트**: 차트를 터치하면 상세 수치(![][image1]\-value 등)가 툴팁으로 노출.

### **\[Step 4\] 최적해 도출 (Optimize)**

* **UI 구성**:  
  * 'Golden Solution' 카드 (금색 테두리).  
  * 최적 인자 조건 리스트 및 예상 ROI 수치 강조.  
* **UX 포인트**: 현재 조건과 최적 조건을 나란히 배치하여 개선 효과(Before/After) 가시화.

### **\[Step 5\] 재현 실험 (Verify)**

* **UI 구성**:  
  * 최적 조건이 입력된 3행의 미니 실험표.  
  * '예측 수율 vs 실제 수율' 오차 막대 그래프.  
* **UX 포인트**: 오차가 5% 미만일 때 '신뢰도 높음' 배지 부여.

### **\[Step 6\] 보고서 생성 (Report)**

* **UI 구성**:  
  * 리포트 썸네일 미리보기.  
  * 하단 \[PDF 다운로드\] 및 \[공유하기\] 버튼.  
* **UX 포인트**: 공유 버튼 클릭 시 WhatsApp, 이메일, 팀즈 등 즉시 연결.

## **4\. 모바일 특화 인터랙션**

1. **Pull-to-Refresh**: 분석 데이터 새로고침.  
2. **Voice Feedback**: 음성 입력 시 "온도 170도 입력되었습니다"라고 음성으로 확인.  
3. **Haptic Feedback**: 버튼 클릭 및 실험 완료 시 진동 알림.
