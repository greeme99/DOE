# DOE AI 웹앱 설계 가이드: 제조 현장 즉시 적용형

**버전**: 1.0 | **대상**: 노코드/로우코드 개발자 | **완성 목표**: 2주 MVP
**목적**: Minitab DOE를 웹에서 5분 자동화, 현장 담당자 생산성 10배 향상

## 🎯 비즈니스 요구사항

```
ROI: 교육생 1인당 월 5공정 × 500만원 = 연 3억원 절감
사용자: 품질/공정 엔지니어 100명 (초보 70%)
접근: 모바일 60%, PC 40% (현장 QR 스캔)
언어: 한국어 (영어 옵션)
```


## 🏗️ 시스템 아키텍처

```
Frontend: React/Vue (PWA 모바일 우선)
Backend: Python FastAPI (Minitab CSV 연동)
DB: PostgreSQL (공정 템플릿 100개)
AI: statsmodels + Plotly (6개 플롯 자동)
Cloud: AWS/GCP (웹앱+API)
```


## 📱 UI/UX 와이어프레임 (모바일 우선)

### **1. 홈: 공정 선택 (30초)**

```
[사출성형 ▼] [식각] [코팅] [화학] [사용자 정의]
↓ 자동 인자 로드
인자1: 온도 [160] [180] ℃
인자2: 압력 [50] [70] bar
목표: 수율 [85%] → [95%]
[20런 생성] [시작] 👈 큰 버튼
```


### **2. 설계표 페이지 (다운로드 즉시)**

```
📥 CSV 다운로드 (Minitab 호환)

Run | A | B | C | Yield 📝
1   |160|50 |10 | [입력]
[실험완료 3/20] 진행률 바
```

## ⚙️ 기능 스펙 (MVP 우선순위)

### **P0: 필수 (Week1)**

```
1. 3인자 20런 표준템플릿 (사출/식각)
2. CSV ↔ Minitab 완벽 호환
3. 5개 핵심 플롯 자동화 (Plotly)
4. Optimizer + 확인실험표
5. PDF 리포트 (ROI 자동계산)
```


### **P1: 고급 (Week2)**

```
6. 4인자 Full/Fractional 자동판단
7. Center P<0.05 → RSM 전환
8. 모바일 QR 입력 (오프라인)
9. 공정 템플릿 10개 (사용자 공유)
10. 부서별 대시보드 (Cpk 추적)
```


## 🔧 백엔드 API 명세

```
POST /api/doe/design
{ "factors":3, "levels":[[160,180],[50,70],[10,15]] }
→ { "runs":20, "design.csv", "random_order":true }

POST /api/doe/analyze  
{ "csv_file":upload, "response":"Yield" }
→ { "pareto":[{name:"A",effect:42,p:0.002}], 
    "plots":[6개 base64], "optimizer":{...} }

GET /api/templates/shotmolding → 3인자 사출 템플릿
```
