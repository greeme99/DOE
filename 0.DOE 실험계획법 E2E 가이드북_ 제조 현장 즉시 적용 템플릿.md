## 3. 3인자 DOE E2E (권장: **표준 템플릿**)

### **3-1. 사출성형 실습 사례**

```
인자: A(온도160/180℃), B(압력50/70bar), C(시간10/15s)
목표: 수율 85%→95%, Cpk 1.45→1.8
예산: 150만원, 기간: 1.5일
```


### **3-2. Minitab 단계별 워크플로 (총 45분)**

#### **Step 1: 설계 생성 (5분)**

```
Stat>DOE>Factorial>Create Factorial Design
Factors: 3, Full factorial, Replicates=2, Center pts=4
→ 20런 설계표 자동생성 + Randomize
```


#### **Step 2: 데이터 입력 후 분석 (20분)**

```
Stat>DOE>Factorial>Analyze Factorial Design
Terms: A,B,C,A*B,A*C,B*C
Graphs: Pareto,Normal,Main Effects,Interactions,Cube 전부
```

**예상 출력**:

```
Pareto: A(42%)>A*B(25%)>B(15%)
Curvature P=0.42 → 선형 OK
```


#### **Step 3: 최적화 (5분)**

```
Stat>DOE>Factorial>Response Optimizer
Yield Target=95 → A=178℃,B=52bar,C=12s (D=0.93)
```


#### **Step 4: 확인실험 (반나절)**

```
Stat>DOE>Factorial>Verify Predictions
3회 실험 → 오차 -0.5% → 승인
```


### **3-3. 핵심 지표 기준**

| 지표 | 목표치 | 미달 시 조치 |
| :-- | :-- | :-- |
| R²adj | >90% | 인자 추가 |
| Curvature P | >0.10 | RSM 전환 |
| D(Desirability) | >0.90 | 수준 조정 |
| 확인 오차 | <5% | 재실험 |

## 4. 4인자 DOE (Full vs Fractional)

### **4-1. Full Factorial (36런, 정밀)**

```
Stat>DOE>Factorial>Create → 4F, Rep=2, Center=4
결과: D=38%,A=22% (Power,Ar flow)
```


### **4-2. 1/2 Fractional (20런, 44% 절감)**

```
Stat>DOE>Factorial>Create → 1/2 fraction of 4F (Res IV)
I=ABCD 확인 → 주효과=3차 → 안전
결과: 동일 D=36%,A=21%
```

**Fractional 판단**:

```
Resolution IV + Pareto 상위2개 주효과 → 95% 신뢰
```

## 5. RSM 곡면 분석 (Center P<0.05 시)

```
Stat>DOE>Response Surface>Create Response Surface Design
Box-Behnken 3F → 15런
Analyze → A²=-1.8 유의 → 175℃ sweet spot
Contour/Surface Plot → 타원 최적영역 식별
```

## 8. AI 코딩 앱 개발 스펙 (노코드/로우코드)

### **8-1. 입력 인터페이스**

```
공정 선택: [사출/식각/코팅/화학] → 자동 인자 템플릿
인자 입력: 이름/저수준/고수준 (3~5개)
목표: Y Target/Upper/Lower
```


### **8-2. 자동 워크플로**

```
1. 인자수 판단 → Full/Fractional 자동선택

4. Optimizer → 최적조건+확인실험표

```
